متغیر

متغیربرای ذخیره داده ها استفاده می شود. هر متغیر دارای نوعی است که نوع داده ای را که در خود نگه می دارد را مشخص می کند متغیرها با استفاده از کلمه کلیدی:var


     var  num  :  int  =  42

ما متغیری به نام num را اعلام کرده ایم و مقدار 42 را به آن اختصاص داده ایم

می توانید یک متغیر را در کد خود مجدداً اختصاص دهید، به عنوان مثال:num = 8
